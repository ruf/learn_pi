编写一个数学考试程序，它可以出题要求学生回答，最后给出分数和答题所耗时间。要求：

1、一共10道数学口算题，格式是 AxB+CxD=? ，其中A B C D都是1到9的随机整数
2、总分100分，答题总时长超过一分钟的扣10%得分，例如答对9道题，但是超时，就只能得9 X 10 X 90% = 81分
3、学生输入如果格式错误，不得分，程序不能因此异常退出




$ python math_test.py
1x8+6x7=? 50
2x9+9x7=? 81
3x1+5x2=? 11
Wrong! 3x1+5x2=13
7x6+6x1=? wawa
Wrong! 7x6+6x1=48

... ... ... ...

You finished this test in 63 seconds, score is 45.
